Catálogos clínicos

Este directorio acepta catálogos oficiales para codificación completa:
- cie11.csv (columnas: key,code,display)
- loinc.csv (columnas: key,code)
- snomed.csv (columnas: key,code)

La app usa estos archivos si existen. En ausencia de archivos, se usan mapas mínimos de ejemplo.

Ejemplo cie11.csv:
key,code,display
ca_prostata,2C82.0,Cáncer de próstata

Ejemplo loinc.csv:
key,code
peso,29463-7

Ejemplo snomed.csv:
key,code
ca_prostata,126906006
